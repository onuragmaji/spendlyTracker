  /model opus

  {

❯ what jinja template in flask                                                  

⏺ Jinja2 is Flask's built-in templating engine. It lets you embed Python-like
  logic in HTML files.

  Basic usage:

  # app.py
  from flask import render_template

  @app.route('/')
  def index():
      return render_template('index.html', name="Anurag", items=[1,2,3])

  <!-- templates/index.html -->
  <h1>Hello, {{ name }}!</h1>

  {% for item in items %}
    <p>{{ item }}</p>
  {% endfor %}

  {% if name == "Anurag" %}
    <p>Welcome back!</p>
  {% endif %}

  Key syntax:

  ┌───────────┬────────────────────────┐
  │  Syntax   │        Purpose         │
  ├───────────┼────────────────────────┤
  │ {{ var }} │ Output a variable      │
  ├───────────┼────────────────────────┤
  │ {% %}     │ Logic (if, for, block) │
  ├───────────┼────────────────────────┤
  │ {# #}     │ Comments               │
  └───────────┴────────────────────────┘

  Template inheritance (most common pattern):

  <!-- templates/base.html -->
  <html>
    <body>
      {% block content %}{% endblock %}
    </body>
  </html>

  <!-- templates/page.html -->
  {% extends "base.html" %}
  {% block content %}
    <p>Page content here</p>
  {% endblock %}

  Your project already uses this — check your templates/ folder.
